Feature Assistant

Note: This extension was created by a Volunteer and is NOT supported by BSA Scoutbook

This extension tool extends the functionality of Scoutbook in various different ways:

Topic Description (links)
PDF Help Documents Installation, Table of Contents, and Troubleshooting
Calendar Repeating Events You can create, edit and delete repeating calendar events
Copy Events You can Copy or Clone Events, with changes as desired, to a new date
Calendar Messaging You can select invitees based on their RSVP status or attendees based on their Attendance status to send them messages
Update Multiple Event Invitees You can add invitees to any number of future events in a quick entry fashion
Enhanced Calendar View You get a modified calendar view that displays a + sign wherever there are multiple events on a single day. This feature will disappear when the display of multiple events is supported on the calendar.
Quick Entry Logs are not available at this time.

Scoutbook has introduced a new Activity log system that is incompatible with the current volunteer created extension. The volunteer developer is investigating the issue and hopes to find a way to restore the feature with the new Activity logs.

Sorry for the inconvenience.
Calendar CSV Event Import You can import calendar events using a CSV file.
Calendar Event CSV Export You can export calendar events to a CSV file.
The RSVP report feature is now built directly into Scoutbook!
Add Scout Birthdays You can add (or delete) scout birthdays to the Unit Calendar with the click of a button.
Payment Log Payment Log Quick Entry You can quick enter credits and debits for groups of Scouts
Payment Log Balance Report You can view/print a Payment Log Balances and Transaction Report for Scouts and optionally the Unit, AND send Balance (Due) Emails to account holders
Payment Log Unit Accounting You can have a Unit Account that works with the Scout Accounts as subaccounts
Swimmer Classification Swimmer Classification Quick Entry You can quick enter swimmer classifications and dates for groups of Scouts and Leaders
Health Record Dates Health Record Quick Entry You can quick enter health record dates for multiple Scouts and Leaders on one screen
OA Membership Data OA Membership Quick Entry You can quick enter OA membership information for multiple Scouts on one screen
OA Membership Report You can view an OA membership report
Session Monitor Popups warn you if a Session Timeout is impending and give you an option to continue the session. A countdown timer is shown at the top of this menu. The extension icon in the browser shows green when logged in, yellow if there has been no activity for 25 minutes, and red after 30 minutes which should coincide with being autmatically logged out by Scoutbook. This feature is limited in Firefox on Android, and will only work well if one Scoubook window is open. An upcoming release of Firefox for Android will enable the full functionality.
Merit Badge Counselor Quick Connect Connect to Merit Badge Counselor Fast Invite Counselors to connect to Scouts without having to mess with Adding Connections
Merit Badge Quick Entry for Units Add Merit Badge Quick Entry You can use Quick Entry to Add merit badges, sign as the Unit Leader, and Assign Merit Badge Counselors
Merit Badge Features for Merit Badge Counselors The Merit Badge Quick Entry for Counselors is now built in to Scoutbook directly! You can use Quick Entry to Approve requirements for multiple scouts in multiple units all at once.
The Merit Badge Req. Quick Entry for Counselors is now built in to Scoutbook directly! You can use Quick Entry to Approve completed Merit badges for multiple scouts in multiple units all at once.
The Quick Pick to Edit Scout Merit Badge is now built in to Scoutbook directly! You can see the Scouts you counsel, narrow the list to a specific badge, and click to edit the Scout's Merit Badge.
MBC Connect to Scouts listed in CSV file Counselors can list multiple Scouts in a CSV file, and import it to connect to them all at once. Useful for a setting up Merit Badge Classes!
Merit Badge Report for Counselors You can get a report for Scouts you counsel, both inside and outside your unit, that shows completion status of the merit badges you counsel.
Merit Badge Import from Summer Camps Black Pug Summer Camp Import You can import Summer Camp Merit Badge Records into Scoutbook from Black Pug Software (scoutingevent.com).
Import Scouts and Parents Scout and Parent CSV Import You can import New Scouts, invite parents, and update their essential profile information from a CSV file.
Parent CSV Import You can import and invite parents for existing Scouts, and update their essential profile information from a CSV file.
Connection Management Preserve parent Connections When changing all Scout permissions on a row in the Connection Manager, the permissions of the adult's children connections will be preserved
Change Scout Column Connections Clicking on the Scout name in the connection manager allows you to change permissions for all adults except parents and Admins
Set Permissions based on Position Set connection permissions between adults and Scouts by setting a default set of permissions based on adult positions
Dashboard-Connection Cleanup for MBCs Merit Badge Counselors can easily identify Scouts that they no longer need a connection to, disconnect from them, which removes unnecessary units from their Dashboard
School Information Update School Information You can easily update school names and grades for multiple Scouts, including a increment grade button
Training Information Quick Enter Scout Training You can easily enter training for multiple Scouts
Adult YPT Report View an Adult YPT training report for leaders AND parents, including status for new 2018 requirements
Messages Send messages based on Patrol or Den You can select message recipients based on patrols and dens
Custom Email Groups You can create and use Custom Email Groups for messaging
As a Sender of messages, you are always copied, even if you forget to click on yourself.
You can preview messages to see what they look like with the BB codes translated
Recipient List and YPT Notice You can include a list of recipient names in a BCC message, and include a YPT notice on messages from or to youth
Roster Diagnostic Roster Analysis A tool for Admins to check for bad parent connections, duplicate accounts, unapproved accounts, and profiles missing councils or districts
Fix parent connections A tool for Admins fix bad connections
Set empty Council and Districts A tool for Admins update profiles missing council and district information
Scout Leader Positions Update Scout Youth Leadership Positions You can easily add and close out Scout Leadership Positions in a Quick Edit
Quick Undo-Reset Requirements Correct requirement marking mistakes Awards, Merit Badges, and Adventures pages now have an option to remove Approvals, Completions, or both.
Extension Upgrade Notifications Look at the bottom of this page. Want to be notified when bugs are fixed? Or when new features introduced? Or not at all... Select your preference.
Council Admin Feature Council User Statistics Council Admins can see recent usage chart by district and unit type
